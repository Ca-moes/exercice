# Weekly Schedule

## Overview

A 6-day Push / Pull / Legs split, each pattern trained **twice per week** (Push 1 / Pull 1 / Legs 1, then Push 2 / Pull 2 / Legs 2). Every session is a short, focused block — **~7 min postural correction + ~20–25 min strength** — done at home. Sunday is full rest.

This replaced the old 3-day plan (Mon/Wed/Fri strength + Tue/Thu CrossFit + running). Training each muscle 2×/week drives more growth than 1×, and splitting the volume across two shorter sessions is what brings each workout back down to ~20–25 min instead of the 40–50 min the old single sessions were taking.

> [!note] Running is paused
> Running is on hold for now (medical). The recovery capacity that used to go to running now comfortably covers the extra strength days. Reintroduce easy runs later — keep hard runs away from leg days when you do.

## How the two sessions differ

Each pattern shares 1–2 **anchor** movements across both days (so you can track progression week to week) and differs on the rest — for variety, complete coverage, and less fatigue stacking. The full split is detailed in each workout note; in short:

| Pattern | Anchors (both days) | Day 1 adds | Day 2 adds |
|---------|--------------------|-----------|-----------|
| Push | Pike push-ups, lean-in lateral raises | Push-ups (loaded) + overhead triceps ext | Deficit push-ups + dips |
| Pull | Inverted rows, a biceps curl | Overhand pull-ups + band pull-aparts | Chin-ups + Y-T-W |
| Legs | Single-leg calf raises | Bulgarian split squats + sissy squats + sliding leg curls | B-stance RDL + hip thrusts + Cossack squats |

## Training Load Notes

- The split is high-frequency but **low per-session volume** — each muscle gets ~4 short sets twice a week. That's easier to recover from than one long brutal session, even at 6 days.
- Sleep and food are the limiter on 6 days/week. If you're run down, the fix is a rest day, not pushing through.
- Keep one genuine rest day (Sunday). If fatigue accumulates mid-week, take a second easy/rest day — frequency is forgiving; chronic under-recovery is not.
- **Groin:** train as normal for now, but scale or substitute the moves flagged in [[leg-workout]] if the right inner-groin area flares. Revisit after the physio visit.
